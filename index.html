KeyCastJS

A minimalist TypeScript library for displaying keystrokes

Try it out!

Type anywhere to see your keystrokes in KeyCastr Svelte mode. Drag to move.

Features:

• Fixed 200×100px compact display in bottom-right corner

• Drag to reposition - automatically anchors to nearest corner

• Stays visible when scrolling, maintains position on window resize

• Up to 6 characters of output with automatic SVG text scaling

• Display clears when modifier keys are pressed/released

• Platform-specific modifier keys (Mac: ⇧⌃⌥⌘, Windows: ⇧^⎇⊞)

• Hold modifiers to see indicators light up below

• Modifier combos appear in display (e.g., ⌘C or ^C)
